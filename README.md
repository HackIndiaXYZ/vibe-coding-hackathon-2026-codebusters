## 📌 Problem Statement

Crores of Indian citizens miss welfare benefits every year — not because they don't qualify, but because:

- Scheme information is **scattered** across dozens of government portals
- Eligibility rules are written in **complex jargon**
- There is **no easy way** to match your personal profile against available schemes
- Application processes are **opaque and intimidating**, especially for first-time applicants

---

## 💡 Solution

**BharatSahayak AI** is an AI-powered welfare navigator that:

- **Checks your eligibility** against 20+ flagship central and state government schemes
- **Explains schemes in plain language** — the way a knowledgeable friend would
- **Guides you step-by-step** through the application process with document checklists
- **Supports multiple languages** including Hindi and English, with voice input
- **Respects your privacy** — works fully in guest mode, with no login required

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Local Development

```bash
# Clone the repository
git clone https://github.com/your-username/sahayak-mitr-ai.git
cd sahayak-mitr-ai

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:3000`.

### Build for Production

```bash
npm run build
npm run start
```
